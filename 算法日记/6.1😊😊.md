### 核心思路
==**双指针滑动窗口**：右指针扩展，左指针收缩，O(n) 找到最短区间。==
### 代码部分

```cpp
int l = 1, cnt = 0, ans_l = 1, ans_r = n;

for (int r = 1; r <= n; r++) {
    b[a[r]]++;
    if (b[a[r]] == 1) cnt++;  // 新增一种画家
    
    while (cnt == m) {  // 包含所有画家，尝试缩左
        if (r - l < ans_r - ans_l) {
            ans_l = l;
            ans_r = r;
        }
        b[a[l]]--;
        if (b[a[l]] == 0) cnt--;  // 少了一种画家
        l++;
    }
}
```

#洛谷p1678 #二分答案 
### 核心思想
**排序 + 二分查找最接近的数**：对每个学生的估分，二分找到第一个 ≥ 估分的分数线，和它前一个位置一起比较，取差值最小的，累加就是答案。
### 代码部分
```cpp
#include <iostream>
#include <algorithm>
#include <cmath>
using namespace std;

typedef long long ll;
int m, n, b;
ll sum = 0;
int a[100050];

int main() {
    cin >> m >> n;
    for (int i = 1; i <= m; i++) {
        cin >> a[i];
    }
    sort(a + 1, a + 1 + m);
    
    for (int i = 1; i <= n; i++) {
        cin >> b;
        
        // 找第一个 >= b 的位置
        int l = 1, r = m;
        while (l < r) {           // ✅ l < r
            int mid = (l + r) / 2;
            if (a[mid] >= b)
                r = mid;
            else
                l = mid + 1;
        }
        
        // l 是第一个 >= b 的位置
        int ans = abs(a[l] - b);
        if (l > 1) {
            ans = min(ans, abs(a[l - 1] - b));
        }
        sum += ans;
    }
    
    cout << sum << endl;
    return 0;
}
```
### 复杂度
O(nlogn)
