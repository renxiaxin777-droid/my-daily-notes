#树状数组
![[Pasted image 20260604234911.png]]
### 问题描述
你要对一段数组进行以下两种操作
1. 在第x个元素上加k
2. 求区间[x,y]内所有元素的和
聪明的你很快想到了双层for循环来实现
但是对于百万个数，这个方法太费时间了，一次操作得等到明年！！
因此，就引入了树状数组这个概念
### 推导过程
聪明的你没有服输，再次想到了把原数组中每两个相邻的元素放在一起，构成一个新数组那么所需的操作就会减半，但是仍然很复杂，那怎么办呢？

首先运用类比的思想，继续向上合成数组，原来有n个数的数组两两相加得到有n/2个元素的数组，再次两两相加，得到含n/4个元素的数组......最终就变成了一个只含一个元素的数组。于是你得到了堆在n个元素上的好多个数组，just like this!
。。。。。。

### 代码部分
```cpp
#include<iostream>
using namespace std;

int n, m;
int a[500050];
long long tree[500050];  // 树状数组

// lowbit函数：获取x的二进制表示中最低位的1所对应的值
int lowbit(int x) {
    return x & (-x);
}

// 单点修改：在位置x上加上k
void add(int x, int k) {
    while (x <= n) {
        tree[x] += k;
        x += lowbit(x);   // 向上跳转，更新所有包含x的节点
    }
}

// 前缀和查询：返回[1, x]的和
long long query(int x) {
    long long res = 0;
    while (x > 0) {
        res += tree[x];
        x -= lowbit(x);   // 向下跳转，累加各个区间
    }
    return res;
}

// 区间查询：返回[l, r]的和
long long range_query(int l, int r) {
    return query(r) - query(l - 1);
}

int main() {
    ios::sync_with_stdio(false);  // 加速输入输出
    cin.tie(0);
    
    cin >> n >> m;
    
    // 初始化树状数组
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
        add(i, a[i]);  // 相当于在第i个位置加上a[i]
    }
    
    // 处理m个操作
    for (int i = 1; i <= m; i++) {
        int op, x, y;
        cin >> op >> x >> y;
        
        if (op == 1) {
            // 操作1：将第x个数加上y
            add(x, y);
        } else {
            // 操作2：输出区间[x, y]的和
            cout << range_query(x, y) << '\n';
        }
    }
    
    return 0;
}
```
### 复杂度分析
O((n+m)logn)
